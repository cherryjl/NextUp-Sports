# NextUp-Sports

An all-in-one sports calendar application that brings together schedules from multiple sports leagues and competitions worldwide using free APIs.

## About

NextUp-Sports is a comprehensive sports calendar web application that allows users to stay updated on upcoming events across their favorite sports. Whether you're into football, basketball, motorsports, hockey, or niche sports like curling and weightlifting, NextUp-Sports keeps you informed about upcoming games and events.

This project was developed as a semester-long group project for **CS 343: Application Development** at James Madison University.

## Features

- **Multi-Sport Calendar**: View upcoming events from 30+ sports leagues and competitions
- **Customizable Appearance**: Toggle dark/light themes to suit your preferences
- **Multiple View Options**: Switch between different calendar views for optimal viewing
- **Background Customization**: Upload custom background images to personalize your experience
- **Language Support**: Multi-language interface for global users
- **Import/Export**: Save and restore your calendar preferences
- **Sports Trivia Minigame**: Test your sports knowledge with our integrated quiz game covering various leagues
- **Print-Friendly**: Print your calendar for offline reference
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Supported Sports

The application includes trivia and calendar integration for:
- **American Sports**: NFL, NBA, MLB, MLS, NHL, NCAA Hockey, UFL
- **European Football**: EPL, La Liga, Bundesliga, Serie A
- **Basketball**: Euroleague, Australian NBL, KBO League
- **Motorsports**: Formula 1, Formula E, IndyCar, MotoGP, Dakar Rally, WRC, Super GT, IMSA, WEC
- **Other Sports**: PGA Golf, LIV Golf, UFC, ONE Championship, Australian NBL, Swedish Hockey, CFL, Cuban Baseball, and more

## Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **APIs**: Multiple free sports data APIs for real-time event information
- **Storage**: Browser localStorage for user preferences
- **Design**: Responsive and customizable

## Features

- **View Events**: Browse the main calendar to see upcoming sports events
- **Change Theme**: Use the appearance toggle to switch between light and dark modes
- **Upload Background**: Personalize your calendar with a custom background image
- **Select Language**: Change the interface language from settings
- **Play Minigame**: Test your sports knowledge with the integrated trivia quiz
- **Export Settings**: Save your preferences to a file for backup
- **Import Settings**: Restore your saved preferences from a file
- **Print Calendar**: Print your calendar view for offline use
